# 通用场景规则

- **两套语言** (ADR-037): 协议术语和用户语言严格分离。场景用户永远不接触协议内部概念（酒店管家比喻：客人说"延迟退房"，你不说"我提交了 PMS 时间戳修改工单"）
- **场景 = 协议投影**: 共用 Protocol API，每个场景通过 SceneAdapter 做翻译
- **浅色主题**: 绝对禁止暗色主题（唯一例外：场景 CLAUDE.md 中有显式授权暗色主题声明）
- **API Key 安全**: 所有前端 API Key 必须走服务端代理，绝不暴露在客户端
