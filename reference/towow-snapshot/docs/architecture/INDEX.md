### D-01 核心结论（2026-02-26）

**标签 = 产品 = 场景 = 应用。** 一个标签就是一个市场/应用。处理深度由场景定义，不需要系统判断。

五个关键发现：
1. **标签=场景=产品**：切换标签 = 切换应用，底层同一协议
2. **端侧 Agent = 用户完整本地 AI**：通爻是 MCP tool，不是沙盒
3. **承诺上链 → 沉淀 → 搜索从响应中涌现**：市场从使用中长出来
4. **面具 × 标签**：两个正交维度，组合覆盖所有场景
5. **协议内置筛选**：HDC 编码 = 内容表示 = 路由地址 = 过滤标准

**待深入**：分布式通信机制（NDN + HDC、Hamming DHT、语义 Gossip）
